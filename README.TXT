PROJECT TITLE: Forerunner Game
PURPOSE OF PROJECT: This was created as the final project for APCS. It is a game where you control a forerunner in a game arena, surviving for as long as possible to collect points while also avoiding obstacles and collecting powerups.
VERSION or DATE: Version 14.0
HOW TO START THIS PROJECT: You can run this project on Eclipse, and this will require the Greenfoot API for it to run.
AUTHORS: Lawrence Cheng, Kevin Ge, David Magnusson
USER INSTRUCTIONS: Enter space to enter the game. Within gameplay, use the arrow keys to move and the space bar to shoot projectiles.
